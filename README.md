# BimGo UI Components

基于 Vue 3 和 Tailwind CSS 的现代化 UI 组件库。

## 组件列表

- [Sidebar](./docs/sidebar.md) - 响应式侧边栏组件
- [Menu](./docs/menu.md) - 可分组的导航菜单组件

## 特性

- 📱 响应式设计
- 🎨 基于 Tailwind CSS
- 💪 TypeScript 支持
- ⚡️ 轻量级
- 🔧 高度可定制
- 📦 开箱即用

## 安装

```bash
npm install @bimgo/ui-components
```

## 使用

```vue
import { Sidebar, Menu } from '@bimgo/ui-components'
```
